# Homebrew Tap

## How do I install these formulae?

`brew install jiulongw/tap/<formula>`

Or `brew tap jiulongw/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "jiulongw/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
